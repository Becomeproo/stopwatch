## ℹ️앱 설명

스톱워치 앱으로 총 4개의 버튼(시작, 정지, 기록, 초기화)으로 구성되어 있다. 추가적으로 특정 시간을 지정하면 해당 시간의 카운트다운을 시작한 뒤 스톱워치를 시작할 수 있는 앱이다.

<br>
<br>

## ✅구현 기능

* 스톱워치 기능
  * 0.1초마다 숫자 업데이트
  * 시작, 일시정지, 정지
  * 정지 전 다이얼로그 알림
* 시작 전 카운트다운 추가
* 카운트다운 3초 전 알림음
* 랩타임 기록

<br>
<br>

## ✅사용되는 기능

* UI
  * ConstraintLayout
  * ProgressBar

* Android
  * AlertDialog
  * Thread
  * runOnUiThread
  * ToneGenerator
  * addView
